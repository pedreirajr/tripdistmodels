# tripdistmodels 0.0.1

## First version:
- Function `furness()` for doubly constrained iterative proportional fitting (IPF) on OD matrices or long tables.
- Function `gravit()` for synthetic doubly constrained gravity modeling with TLFD calibration and support for multiple friction functions.
- Unit tests for `furness()` and `gravit()`.
- Set up GitHub Actions CI for R CMD check.
